Nécessaire pour jouer : 
2 manettes
au moins un copain rigolo

commandes : 
	menu :
		on rejoint la partie avec start, le joueur 1 lance la partie avec A
	game : 
		stick gauche pour se déplacer
		stick droit pour orienter son viseur
		gachette droite pour tirer

principe du jeu
	vous tirez des balles blanches qui prennent votre couleur lorsqu'elles rencontrent un mur
	si une balle coloré rencontre un jouer d'une autre couleur, elle prendra sa couleur
	une balle blanche peut tuer n'importe quel joueur
	une balle coloré ne peut tuer que le joueur de la même couleur
	le dernier joueur a avoir des viez gagne

Have fun
